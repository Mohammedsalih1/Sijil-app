قم بتحويل تطبيق سِجِل المبني باستخدام React + Vite + Tailwind CSS إلى Progressive Web App (PWA) احترافي يعمل Offline بشكل كامل.

المتطلبات:

## الهدف

أريد أن يعمل التطبيق كتطبيق جوال حقيقي يمكن تثبيته على Android و iPhone ويعمل بدون اتصال بالإنترنت.

---

## تثبيت PWA

قم بإضافة جميع الإعدادات اللازمة لتحويل التطبيق إلى PWA.

يشمل ذلك:

* Manifest.json
* Service Worker
* App Icons
* Offline Support
* Install Prompt

---

## استخدام أفضل مكتبة

استخدم:

vite-plugin-pwa

وقم بإعدادها بشكل احترافي داخل Vite.

---

اسم التطبيق:

سِجِل

---

## أيقونات التطبيق

أضف جميع الأحجام المطلوبة:

192x192

512x512

Maskable Icons

وقم بإعدادها داخل Manifest.

إذا لم تكن الأيقونات موجودة فقم بإنشاء placeholders واضحة مع تعليق يوضح مكان استبدالها بالشعار النهائي.

---

## Offline Support

أريد أن يعمل التطبيق بالكامل بدون إنترنت.

يجب تخزين الملفات الأساسية داخل Cache:

* HTML
* CSS
* JavaScript
* Icons
* Fonts

حتى يتم فتح التطبيق بدون إنترنت.

---

## Local Storage

التطبيق يعتمد على Local Storage.

يجب التأكد من أن:

* إضافة العمليات تعمل Offline.
* تعديل العمليات يعمل Offline.
* حذف العمليات يعمل Offline.
* البحث يعمل Offline.
* الإحصائيات تعمل Offline.
* نظام التفعيل يعمل Offline.

---

## شاشة عدم وجود اتصال

إذا لم يكن هناك اتصال بالإنترنت:

لا تمنع استخدام التطبيق.

أظهر فقط تنبيهاً صغيراً:

"أنت تعمل حالياً بدون اتصال بالإنترنت"

ويختفي تلقائياً.

---

## Install Prompt

أضف زر تثبيت التطبيق.

إذا كان التطبيق غير مثبت:

اعرض زر:

"تثبيت التطبيق"

وعند الضغط عليه:

قم بإظهار نافذة تثبيت PWA.

service-worker

PWA configuration

وأي ملفات أخرى مطلوبة.

---

## مهم جداً

لا تقم بتغيير أي منطق موجود داخل التطبيق.

لا تقم بتغيير Local Storage.

لا تقم بتغيير نظام الفترة التجريبية.

لا تقم بتغيير نظام أكواد التفعيل.

قم فقط بإضافة دعم PWA و Offline Mode بشكل احترافي مع الحفاظ على جميع وظائف التطبيق الحالية.
